# Quiz app

Markdown style quiz generator. A custom Markdown language that includes custom commands for creating HTML.

```markdown
# Quiz

Litt teori om Uncertainty...

[quiz
? P(A,B)=P(A|B)*P(B)
- ja
+ nei
]
[quiz
? KAtter
+ ja
- nei
+ kanskje
]
```
